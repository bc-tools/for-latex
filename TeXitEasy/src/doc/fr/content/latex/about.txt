==================================
À propos de la distribution ¨latex
==================================

La fonction ``about`` renvoie dans un dictionnaire des informations nécessaires pour utiliser la fonction ``install`` qui permet d'installer localement un package personnel.
Voici un exemple d'utilisation sur un ordinateur ¨mac avec la distribution ¨texlive installée via ¨mactex (la clé ``'localdir'`` est le chemin du dossier où installer les packages personnels). Nous utilisons ``pprint.pprint`` pour faciliter la lecture du dictionnaire.

pyterm::
    >>> from pprint import pprint
    >>> from mistool import latex_use
    >>> pprint(latex_use.about())
    {'latexfound': True,
     'latexname': 'texlive',
     'localdir': PosixPath('/usr/local/texlive/texmf-local'),
     'osname': 'mac'}
