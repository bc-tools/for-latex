===============================
Nettoyage des fichiers spéciaux
===============================

Une compilation ¨latex produit toujours des fichiers temporaires que l'on ne souhaite pas toujours garder. Comment les nettoyer ? La fonction ``clean`` a été conçue pour ce genre de situation.
Voici un exemple d'utilisation où nous utilisons la classe ``term_use.DirView`` pour afficher les ¨chgts dans le dossier path::``/Users/projetmbc/latex`` du fichier ¨latex path::``file.tex`` pour lequel nous voulons faire un peu de ménage.

pyterm::
    >>> from mistool.latex_use import clean
    >>> from mistool.os_use import PPath
    >>> from mistool.term_use import DirView
    >>> latexdir = PPath("/Users/projetmbc/latex")
    >>> print(DirView(latexdir.parent).ascii)
    + latex
        * file.aux
        * file.log
        * file.pdf
        * file.synctex.gz
        * file.tex
    >>> clean(ppath = latexdir, showinfos = True)
    * Cleaning for "/Users/projetmbc/latex/file.tex"
    >>> print(DirView(latexdir.parent).ascii)
    + latex
        * file.pdf
        * file.tex


Dans cet exemple, nous avons utilisé ``showinfos = True`` afin de savoir ce qui a été nettoyé (ceci est indispensable si vous nettpyez un gros dossier). Si cela ne vous intéresse pas, il suffit d'utiliser ``clean(latexdir)``.


L'argument ``ppath`` peut aussi correpondre au chemin d'un dossier. Dans ce cas, tous les fichiers d'extension path::``tex`` vont être cherchés, et à chaque les fichiers spéciaux associés seront supprimés.


info::
    Dans notre exemple, les extensions des fichiers à détruire sont celles indiquées dans la liste ``EXTS_TO_CLEAN`` qui est la valeur par défaut de l'argument ``exts``. La section suivante explique comment utiliser votre propre liste pour par exemple garder certains fichiers spéciaux.
