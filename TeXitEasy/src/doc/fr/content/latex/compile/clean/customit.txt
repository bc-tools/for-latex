==============================================
Choisir les extensions des fichiers à nettoyer
==============================================

La fonction ``clean`` utilise par défaut l'argument ``exts`` égal à la liste ``EXTS_TO_CLEAN`` pour connaître les extensions des fichiers spéciaux à nettoyer. Voici comment est définie cette constante
((
    Le code du fichier path::``mistool.config.latex`` comporte quelques commentaires complémentaires pour chaque rangement thématique.
)).

python::
    ...
        input = ¨content/extstoclean.py


Par exemple, si dans le dossier path::``/Users/projetmbc/latex`` on souhaite garder les fichiers spéciaux d'extensions path::``synctex.gz`` et path::``log``, on pourra procéder comme suit.

python::
    from mistool.latex_use import clean, EXTS_TO_CLEAN
    from mistool.os_use import PPath

    myexts = [
        x for x in EXTS_TO_CLEAN
        if x not in ['log', 'synctex.gz']
    ]

    latexdir = PPath("/Users/projetmbc/latex")

    clean(ppath = latexdir, exts = myexts)


En fait la liste ``EXTS_TO_CLEAN`` est produite à partir de ``TEMP_EXTS`` qui est le dictionnaire défini comme suit où les clés des rangements thématiques sont identiques à celles utilisées pour commenter le code de la liste ``EXTS_TO_CLEAN``.

python::
    ...
        input = ¨content/tempexts.py


``EXTS_TO_CLEAN`` et ``TEMP_EXTS`` peuvent vous aider à produire vos propres listes d'extensions pour des fichiers à retirer.
