Le package \tdocpack{hyperref} est importé en coulisse avec les réglages ci-dessous.

  colorlinks,
  citecolor = \tdoclinkcolor,
  filecolor = \tdoclinkcolor,
  linkcolor = \tdoclinkcolor,
  urlcolor  = \tdoclinkcolor
}
