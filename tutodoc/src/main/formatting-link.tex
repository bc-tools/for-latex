\subsection{Liens dynamiques}

Le package \tdocpack{hyperref} est importé avec juste les réglages suivants où \tdocinlatex{tutodoc@link@color} est une couleur définie par le thème choisi.


\begin{tdoclatex}[code]
\hypersetup{
  colorlinks,
  citecolor = tutodoc@link@color,
  filecolor = tutodoc@link@color,
  linkcolor = tutodoc@link@color,
  urlcolor  = tutodoc@link@color
}
\end{tdoclatex}
