\section{Choisir la langue au chargement du package}

La présente documentation utilise le français via \docinlatex|\usepackage[lang = FR]{bdoc}| .
Pour le moment, on a juste deux choix.

\begin{enumerate}
    \item \docinlatex|EN|, valeur par défaut, est pour \docinEN*{\docprewhy{EN}{glish}} soit \docinEN{anglais}.

    \item \docinlatex|FR| est pour \docinEN*{\docprewhy{FR}{ench}} soit \docinEN{français}.
\end{enumerate}
