\section{Choisir la langue au chargement du package}

La présente documentation utilise le français via \bdocinlatex|\usepackage[lang = fr]{bdoc}| .
Pour le moment, on a juste deux choix.

\begin{enumerate}
    \item \bdocinlatex|en|, valeur par défaut, est pour \bdocinEN*{\bdocprewhy{en}{glish}} soit \bdocinEN{anglais}.

    \item \bdocinlatex|fr| est pour \bdocinEN*{\bdocprewhy{fr}{ench}} soit \bdocinEN{français}.
\end{enumerate}
