\bdocexa[Numéro et date d'une version]

L'exemple suivant montre une macro plus complète que la précédente mais avec un fonctionnement similaire
\footnote{
    La macro \bdocmacro{docnew} a été conçue pour la documentation elle-même et \bdocmacro{docversion} juste pour l'historique des changements.
}.

\inputbdoclatexreal{examples/version-n-change/version.tex}


\begin{bdocrem}
    \leavevmode

    \begin{enumerate}
        \item Le numéro de version sera du type \texttt{maj.min.bug} ou \texttt{maj.min.bug-extra} avec trois entiers \texttt{maj}, \texttt{min}, \texttt{bug} et un optionnel \texttt{extra} égal à \texttt{alpha}, \texttt{beta} ou \texttt{rc} pour \bdocquote{release candidate}.

        \item Noter dans le rendu final le format français \texttt{JJ/MM/AAAA} de la date alors que dans le code celle-ci doit être donnée au format anglais \texttt{AAAA-MM-JJ}.
    \end{enumerate}
\end{bdocrem}
